# Mimer Tenant Portal

## Getting started

1. Install nvm (on Mac: `brew install nvm`)

### Starting the backend

1. cd into `packages/backend`
2. `nvm use`
3. `npm install`
4. `npm run dev`

### Starting the frontend

1. cd into `packages/frontend`
2. `nvm use`
3. `npm install`
4. `npm run dev`
5. Point your browser to http://localhost:5000
